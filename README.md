TP de la Materia

## ¿Cómo correr este proyecto?

En la carpeta, usando `cmd.exe`, tirás:

```sh
npm install
npm start
```

Va a aparecer un código QR en la consola de `cmd.exe`. Te bajás la app "Expo" en tu celular, scaneás ese código QR, y listo, tenés la app andando.
